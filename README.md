# Prueba Técnica - Frontend

Este proyecto es una aplicación frontend construida con React y Vite. Utiliza Material-UI para los componentes de la interfaz de usuario y herramientas como Formik para la gestión de formularios y Axios para la realización de solicitudes HTTP.

## Características Principales

 - React 18: Para la construcción de interfaces de usuario interactivas.
 - Vite: Herramienta de desarrollo y construcción rápida.
 - Material-UI (MUI): Librería para fácilitar la maquetación del proyecto y manejar un theme. 
 - Formik: Manejo de formularios.
 - Axios: Librería para hacer peticiones HTTP.

## Requisitos Previos
Se debe tener instalados los siguiente programas:

- Node.js (>= 14.x)
- npm o yarn

## Instalación

1. Clonar el repositorio:


```bash
git clone https://github.com/usuario/prueba-tecnica.git

```
2. Instalar las dependencias:

```bash
cd prueba-tecnica
npm install

```
3. Iniciar el proyecto en modo desarrollo:

```bash
cd prueba-tecnica
npm run dev

```
La aplicación estará disponible en http://localhost:5173.

## Scripts Disponibles

En el proyecto puedes ejecutar los siguientes scripts:

  -	npm run dev: Inicia la aplicación en modo desarrollo.
  - npm run build: Construye el proyecto para producción.
  - npm run preview: Previsualiza la aplicación después de la construcción.
  - npm run lint: Ejecuta ESLint para analizar el código en busca de errores de estilo o sintaxis.
